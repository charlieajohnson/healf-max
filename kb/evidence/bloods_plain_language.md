# Summary

Blood markers should be explained in normal language: what the marker means, why it may matter, what to do next, and what not to infer.

# Use when

Use when a customer mentions results or asks what a marker means. Biomarker context should shape routing, not create a shopping list.

# Useful phrasing

Start with the bottleneck and explain the signal without turning the answer into a stack.

# Boundary

This record supports wellness education and category routing only. It does not establish diagnosis, treatment or dosing.
